# Divani Mods v1.2.8
New Roles and Modifier + Duelist option + adjustments
## Added

### Added Role: Retributionist (Crewmate Killing)

When you get killed, you spawn on a random vent as the Vengeful Soul and you get a limited time to find and kill your killer. 
If you succeed, you get to live again.
If you fail, you become a normal ghost. 
Your killer cannot vent or report and Concealing impostors and the Glitch cannot use their Buttons

### Added Role: Clockstopper (Crewmate Support)

Finish a set amount of tasks to reset kill- and ability-button cooldowns of Impostors and a configured set of Neutral roles.

### Added Modifier: UAV: (Univeral Utility)

Call in a UAV: while active, open the map to see everyone walking around.
If configured, teammates can see this effect too!
Enemy teams are alerted about an active enemy UAV


## Bugfixes:

### Duelist

- Fixed a bug where a teleport-back causes a player to get stuck in walls
- Fixed a bug where trapper (Revealer-TOUE) recorded players in Duel.

## Role/Modifier Changes:

### Duelist

- Added option to to configure Duel start spawns:
1. Close (Unchanged default)
2. Far (Two set points on each map far away from each other)
3. Random (Gamble, Gamble)
- Disable buttons of players returning to the ship
- When a duel ends, players get a red/green flash if they lose/win
